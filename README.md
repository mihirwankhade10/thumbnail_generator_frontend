# Frontend - Video Thumbnail Generator

This is the frontend application for the Video Thumbnail Generator project, built with React and Vite.

## Tech Stack

- **React**: UI library.
- **Vite**: Modern frontend build tool.
- **Axios**: For making API requests to the backend.
- **React Router**: For client-side routing.
- **Vanilla CSS**: For styling.

## Running the Frontend

1. Navigate to the frontend directory:
   ```bash
   cd thumbnail_generator_frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

The application will be running on `http://localhost:5173` (or the port specified by Vite).
